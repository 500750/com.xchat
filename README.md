## 适配版本
- [x] 8.0.55

### 食用方法:
- 下载apk，安装
- lsposed 激活模块，作用域勾选微信
- 强制重启wechat
- 长按 ”我的“ -> "设置" 选项 可呼出设置面板，目前较为简陋...🤷🏻‍♀️
